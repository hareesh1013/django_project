Student Management System
Project Goal

A web app where users can:

Register/Login

Add, view, edit, delete students

Use templates and Bootstrap

See authentication in action


Step 1: Setup Project


# Install Django
pip install django

# Create project
django-admin startproject student_mgmt

cd student_mgmt

# Create app
python manage.py startapp students

# Run server
python manage.py runserver


Explanation:

student_mgmt/ → main project folder

students/ → app where models/views/templates go

runserver → starts dev server at http://127.0.0.1:8000
